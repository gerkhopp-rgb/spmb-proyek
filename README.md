## 🚀 Cara Menjalankan

### Lokal (VS Code)
1. Clone repository ini:
   ```bash
   git clone https://github.com/username/smk-budi-bakti.git
   ```
2. Buka folder di VS Code
3. Install ekstensi **Live Server**
4. Klik kanan `index.html` → **Open with Live Server**

### Online (GitHub Pages)
1. Buka repository di GitHub
2. Klik **Settings** → **Pages**
3. Pilih branch `main` → folder `/ (root)` → **Save**
4. Website aktif di `https://username.github.io/smk-budi-bakti`
